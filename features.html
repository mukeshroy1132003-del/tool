Full feature set

Every layer of your stack, watched.

From a single ping check to distributed tracing across microservices — PulseGrid scales with what you're running.

01 — Uptime & synthetic monitoring

Catch it before a customer does.

Simulated checks run continuously from our global network so you see problems before real users do.

HTTP(S) & API monitoring

Status code, response time, keyword, and JSON-path assertions on every check.

DNS & SSL monitoring

Certificate expiry warnings and DNS record change alerts, weeks in advance.

Port & protocol checks

FTP, SMTP, POP, IMAP, and raw TCP/UDP port monitoring.

Multi-step transactions

Script full user flows — login, add to cart, checkout — and alert on any broken step.

Global vantage points

Run checks from 100+ locations to catch region-specific outages and CDN issues.

Maintenance windows

Silence alerts during planned deploys without turning monitoring off entirely.

02 — Infrastructure monitoring

Servers, containers, and the network between them.

Agent-based metrics for the machines your application actually runs on.

Linux & Windows servers

CPU, memory, disk I/O, and per-process resource usage.

Containers & Kubernetes

Pod, node, and cluster-level health with auto-discovery.

Network devices

SNMP polling for switches, routers, and firewalls.

Database monitoring

Query performance and connection health for popular databases.

Custom metrics

Push any metric from your own scripts via a simple API.

Capacity forecasting

Trend lines that flag when you'll run out of headroom.

03 — APM & real user monitoring

From the server, to the code, to the browser.

Follow a single request end-to-end — including the parts your users actually feel.

Distributed tracing

See every hop a request makes across services, with timing at each stage.

Code-level diagnostics

Slow method and query detection down to the exact line.

Real user monitoring

Actual page-load and interaction timing, segmented by browser, device, and region.

Error tracking

Group and prioritize exceptions by frequency and user impact.

Core Web Vitals

Track LCP, INP, and CLS trends over time for every page.

Session replay

Reconstruct the exact sequence of clicks that led to an error report.

04 — Logs & AIOps

Find the one line in a million.

Centralized log management

Ingest, parse, and search structured or unstructured logs from every service in one place.

Anomaly detection

Machine-learning baselines flag unusual spikes before they become incidents, and correlate root causes across metrics, logs, and traces.
